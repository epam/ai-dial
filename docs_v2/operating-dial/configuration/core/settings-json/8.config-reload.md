# Config Reload

Config reload settings in `settings.json` control how DIAL Core loads and merges dynamic configuration files ([`config.json`](../config-json/index)).

### `config.files`

List of paths to dynamic settings files. Multiple files are merged in order.

**Default:** `aidial.config.json`

```json
{
  "config": {
    "files": ["/opt/config/config.json"]
  }
}
```

### `config.reload`

Config reload interval in milliseconds.

**Default:** `60000` (1 minute)

```json
{
  "config": {
    "reload": 30000
  }
}
```

### `config.jsonMergeStrategy.overwriteArrays`

When `true`, arrays are overwritten during merge; when `false`, they are concatenated.

**Default:** `false`

```json
{
  "config": {
    "jsonMergeStrategy": {
      "overwriteArrays": true
    }
  }
}
```
