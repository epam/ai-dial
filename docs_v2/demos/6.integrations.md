# Integrations

DIAL's API-first design and Unified API make it straightforward to connect to the tools your teams already use. These demos show DIAL embedded in code editors, productivity apps, automation platforms, and model providers.

## AI code assistant (Continue)

Configure [Continue](https://www.continue.dev/), an open-source coding assistant, against DIAL — an example of how the Unified API and standards compatibility let community tools integrate quickly.

https://youtu.be/kK8gtZBdzr4

## Microsoft Copilot

Connect DIAL applications to Microsoft Copilot Studio with a custom connector built from DIAL's OpenAPI definition, including API key authentication and live queries against StatGPT.

https://youtu.be/kzOu7CPnflM

## Microsoft Excel

The DIAL Excel plug-in calls DIAL models and applications from within a spreadsheet, constructing prompts from other cells and comparing answers across models.

https://youtu.be/ifPvHvnWgAc

## n8n

Integrate DIAL with the n8n automation platform to build agentic workflows — demonstrated with an AI-powered HR assistant that responds to employee emails automatically.

https://youtu.be/uG_ed7ypfcs

## NVIDIA NIM

Deploy, manage, and use AI models from NVIDIA NIM in a DIAL environment — the full workflow from model selection to end-user access.

https://youtu.be/WQjv9r5sVuk
