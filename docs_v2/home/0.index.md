# Welcome to DIAL

DIAL is an open-source, enterprise-grade platform that gives you a single gateway to language models, a place to build and run AI applications, and the access control, observability, and collaboration features needed to operate them in production. This page routes you to the right starting point for your role.

New to the platform? Read [What is DIAL](../understand-dial/positioning/0.what-is-dial.md) for the big picture, then return here to pick a path.

## Choose your path

Each path below is a focused quick start. Pick the one that matches what you want to do today.

|---|---|---|
| **Developer** | Run DIAL locally and chat with your first application | [Developer quick start](./developer-quick-start.md) |
| **DevOps / Platform engineer** | Deploy DIAL and call it through the Unified API | [DevOps quick start](./devops-quick-start.md) |

| **End user** | Use DIAL Chat day to day | [End user guide](./end-user-guide.md) |
