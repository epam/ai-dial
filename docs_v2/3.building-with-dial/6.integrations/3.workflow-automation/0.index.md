# Workflow automation

Workflow automation tools let you chain applications, services, and APIs into pipelines that run without manual steps. This section shows how to make DIAL a node in those pipelines, so an automated workflow can call DIAL's models and applications alongside every other system it touches. It is for app developers connecting DIAL to low-code automation builders.

## Available integrations

- [Integration with n8n](1.n8n.md) — build a custom n8n node that calls the DIAL [Unified API](https://dialx.ai/dial_api) for chat completions and text embeddings, then use it in your workflows
