# Integrations

DIAL is an open orchestration platform that works alongside the tools your organization already uses. This section is for developers who want to surface DIAL's models, applications, and Tool Sets inside external systems — without rebuilding those experiences from scratch. Integrations are grouped by purpose so you can find the pattern that matches your goal.

## Chatbot integrations

AI embedded in messaging and conversation platforms, so users reach DIAL from where they already chat.

[Go to Chatbot integrations](1.chatbot-integrations/0.index.md)

## Productivity add-ins

AI embedded in desktop and productivity tools, such as spreadsheet functions that call DIAL.

[Go to Productivity add-ins](2.productivity-add-ins/0.index.md)

## Workflow automation

DIAL as a node in automation pipelines, callable from low-code workflow builders.

[Go to Workflow automation](3.workflow-automation/0.index.md)

## Agentic tools

DIAL as the model backend for agentic coding tools, so command-line agents and IDE assistants run their turns on DIAL.

[Go to Agentic tools](4.agentic-tools/0.index.md)

## Orchestration patterns

Routing and coordination across providers and models within DIAL — failover, cost-capped routing, and model A/B testing.
