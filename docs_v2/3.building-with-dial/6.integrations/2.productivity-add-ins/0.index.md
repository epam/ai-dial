# Productivity add-ins

Productivity add-ins bring DIAL's models and applications into the desktop and productivity tools your users already work in, such as spreadsheet functions that call DIAL directly from a cell. Instead of copying data into a separate chat window, users invoke DIAL where their work already lives. This section is for app developers connecting DIAL to productivity software.

## Available integrations

- [Integration with Microsoft Excel](1.ms-excel.md) — install the DIAL Add-in for Microsoft Excel and call DIAL models from spreadsheet formulas with `Dial.ASK` and `Dial.LISTMODELS`
