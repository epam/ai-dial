# Mind Map Studio overview

Mind Map Studio is a DIAL application type for creating interactive knowledge graphs from documents and URLs. It transforms source material into a navigable visual graph where users can explore relationships between concepts through both a graphical interface and natural-language conversation. Responses are grounded in the provided sources — the system does not generate information beyond what the sources contain.

This page explains what Mind Map Studio does, how it works, and when it is the right tool.

## How Mind Map Studio works

Mind Map Studio follows a three-stage workflow:

### 1. Provide sources

You supply the raw material for the knowledge graph. Mind Map Studio accepts:

- **Web URLs** — public web pages.
- **PDF files** — uploaded documents.
- **Combinations** — mix URLs and PDFs in a single graph.

### 2. Generate the knowledge graph

Mind Map Studio analyzes the sources and builds a knowledge graph — a network of nodes (concepts) and edges (relationships). The generation process extracts entities, identifies how they relate, and structures them into a navigable graph.

The resulting graph is **deterministic**: it represents only information found in the provided sources. The accompanying chatbot answers questions based on the graph content and will not fabricate information beyond what the sources contain.

### 3. Customize and publish

After generation, you can refine the graph through Mind Map Studio's editor:

- **Content editing** — add, remove, or modify nodes and connections in graph view or table view.
- **Theme customization** — style the graph and chatbot using the visual editor or JSON editor. Import and export custom themes.
- **Preview** — test the graph and chatbot in a live preview before publishing.

When ready, save the app and share or publish it through the standard DIAL workflow.

## What Mind Map Studio produces

A Mind Map Studio app provides two interfaces to end users:

- **Visual graph** — an interactive node-and-edge visualization. Users can click nodes to explore related concepts, zoom, pan, and navigate the information structure.
- **Conversational chatbot** — a natural-language interface grounded in the graph content. Users ask questions and receive answers derived from the source material.

Both interfaces work together: clicking a node can inform the chatbot context, and chatbot answers can highlight relevant graph sections.

## Use cases

Mind Map Studio is effective for:

- **Research exploration** — visualize relationships in academic papers, technical documentation, or market research.
- **Document analysis** — extract and navigate the structure of large documents or document collections.
- **Onboarding materials** — create interactive knowledge bases for new team members or customers.
- **Knowledge base visualization** — turn flat documentation into an explorable graph.
- **Public-facing knowledge portals** — the [DIALX website home page](https://dialx.ai/) is built using Mind Map Studio.

## Deployment and access

Mind Map Studio is a schema-based application type managed by the DIAL platform:

- **Platform-hosted** — the platform handles deployment and scaling. No infrastructure management required.
- **Created through DIAL Chat** — use the Application Builder wizard to create Mind Map apps.
- **Standard sharing model** — private by default, shareable with specific users, publishable to the Marketplace.

## Limitations

- Sources are provided at creation time. The graph does not update automatically when source content changes — you must regenerate with updated sources.
- The system is designed for structured information exploration, not free-form conversation. It deliberately limits responses to source-grounded content.
- Graph quality depends on source quality. Poorly structured or ambiguous source material produces less useful graphs.
