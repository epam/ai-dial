# Chat customization

DIAL Chat is the default web UI for interacting with models and applications on the DIAL platform. This section covers the developer-facing extension points and customization options that let you control what Chat renders, how it looks, and how it integrates with your applications.

## What's in this section

- [**Custom content in Chat**](1.custom-content.md) — Reference for the `custom_content` API: attachments, stages, Markdown, visualizers, and Plotly.
- [**Data visualization**](2.data-visualization.md) — How to render charts and visual content in DIAL Chat using Plotly and Markdown.
- [**Create a custom visualizer**](3.create-custom-visualizer.md) — Step-by-step tutorial for building a custom visualizer with the ChatVisualizerConnector library.
- [**Chat localization**](4.chat-localization.md) — How to customize and translate DIAL Chat UI text using the i18n framework.
- [**Theming and design**](5.theming-and-design.md) — How to customize the Chat layout, color scheme, and branding.
- [**Marketplace**](6.marketplace.md) — What DIAL Marketplace is and how it serves as a hub for applications, models, and collaboration.
- [**Create a custom theme**](7.create-custom-theme.md) — Step-by-step tutorial for setting up the themes server with a custom color scheme and logo.
- [**DIAL Overlay**](8.dial-overlay.md) — How DIAL Overlay embeds Chat in external web applications via iframe, with architecture, configuration, and API reference.
- [**Embed Chat with DIAL Overlay**](9.embed-chat-in-web-app.md) — Step-by-step tutorial for integrating DIAL Chat into a host page using the Overlay library.
