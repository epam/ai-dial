# Demos

These are short video showcases of what DIAL can do. Each is a few minutes long, requires no setup, and answers a single question: *what does this capability look like in practice?* They are for evaluators and decision-makers who want to see the platform in action before reading how to build with it.

There is no code and no installation here. When you are ready to build, the [Building with DIAL](../3.building-with-dial/0.index.md) tutorials and the [Use Cases](../8.use-cases/0.index.md) scenarios pick up where these videos leave off.

## Browse the demos

- [Platform overview](1.platform-overview.md) — what DIAL is and the one API behind it.
- [Chat capabilities](2.chat-capabilities.md) — the end-user experience: chat, marketplace, collaboration, and sharing.
- [Apps and agents showcase](3.apps-and-agents.md) — real applications built on DIAL, from RAG to multi-agent assistants and mind maps.
- [Deployment and admin walkthrough](4.deployment-and-admin.md) — standing DIAL up in the cloud, locally, and running it day to day.
- [Developer features](5.developer-features.md) — extension points and developer capabilities: buttons, interceptors, access control, evaluation.
- [Integrations](6.integrations.md) — DIAL inside code editors, Office, automation platforms, and model providers.
