# Deployment and admin walkthrough

What it takes to stand DIAL up and run it. These demos show the Admin Panel, one-click marketplace deployments, and quick local setups with self-hosted models — enough to picture operating DIAL before you read the step-by-step guides.

## DIAL Admin Panel

The Admin Panel gives administrators a single interface to configure, manage, and monitor the DIAL ecosystem.

https://youtu.be/iLUb3I6xuBA

## Deploy from AWS Marketplace

A walkthrough of deploying DIAL from the AWS Marketplace — a quick test installation to see the platform running in your own cloud.

https://youtu.be/vE6mLwR6IGM

## Deploy from Google Cloud Marketplace

Deploying DIAL from the Google Cloud Marketplace, which provisions the service accounts and a VM pre-configured with Vertex AI models.

https://youtu.be/PV02IUqLo1c

## Deploy DIAL with Ollama

Stand up DIAL from GitHub with DIAL Chat and self-hosted vision models from Ollama, walking through the configuration files that connect models and applications.

https://youtu.be/uRUzvSmtkeI

## Deploy DIAL with DeepSeek R1

Set up a fully open-source GenAI environment on your own machine using DIAL and DeepSeek R1.

https://youtu.be/79jR9-wg4vk
