# Encryption

Encryption settings configure AES encryption for blob storage prefix hashing.

### `encryption.secret`

AES encryption secret for blob storage prefix hashing.

**Required**

```json
{
  "encryption": {
    "secret": "your-secret-value"
  }
}
```

### `encryption.key`

AES encryption key for blob storage prefix hashing.

**Required**

```json
{
  "encryption": {
    "key": "your-key-value"
  }
}
```

In Docker Compose deployments, these values are typically injected from the `.env` file:

```json
{
  "encryption": {
    "secret": "${DIAL_CORE_ENCRYPTION_SECRET}",
    "key": "${DIAL_CORE_ENCRYPTION_KEY}"
  }
}
```

:::note
In production, use a secrets manager instead of plaintext values. See the [.env reference](../../docker-compose/env-reference) for variable definitions.
:::
