# DIAL n8n Integration

https://youtu.be/uG_ed7ypfcs

Learn how to integrate DIAL's open-source agentic AI platform with n8n (nodemation) to create powerful automated workflows for your enterprise.

In this technical tutorial, we demonstrate building an AI-powered HR assistant that automatically responds to employee emails using custom DIAL agents and n8n automation.

> Refer to the [DIAL n8n Integration Guide](/docs/tutorials/1.developers/5.integrations/3.n8n-integration.md) for detailed instructions on how to integrate DIAL with n8n via a custom node.
