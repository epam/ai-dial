# Toolsets Security & Miscellaneous

This page covers toolsets security settings, invitation TTL, per-request API keys, and other miscellaneous static settings in `settings.json`.

## Toolsets security

### All OAuth settings

```json
{
  "toolsets": {
    "security": {
      "allowedRedirectUris": ["https://chat.example.com/callback"],
      "resourceSchema": "https",
      "resourceHost": "dial.example.com",
      "scopesSupported": ["openid", "profile", "email"]
    }
  }
}
```

### `toolsets.security.allowedRedirectUris`

Allowed redirect URIs for OAuth sign-in.

**Default:** `[]`

### `toolsets.security.resourceSchema`

URL schema for resource identifier (RFC 9728).

**Default:** `https`

### `toolsets.security.resourceHost`

Public hostname of the resource server for token validation.

### `toolsets.security.scopesSupported`

OAuth 2.0 scope values for authorization requests.

### KMS and encryption settings

For toolset credential encryption:

#### All KMS and encryption settings

```json
{
  "toolsets": {
    "security": {
      "kms": {
        "provider": "aws",
        "keyId": "arn:aws:kms:us-east-1:123456789012:key/example-key-id",
        "region": "us-east-1"
      },
      "encryption": {
        "algorithm": "AES",
        "keySize": 256,
        "cipherTransformation": "AES/GCM/NoPadding",
        "ivLengthBytes": 12,
        "gcmTagLengthBits": 128
      }
    }
  }
}
```

#### `toolsets.security.kms.provider`

KMS provider: `aws`, `azure`, `gcp`, or `unencrypted`.

**Default:** `unencrypted`

#### `toolsets.security.kms.keyId`

KMS key identifier.

#### `toolsets.security.kms.region`

KMS geo region. Required for `aws`.

#### `toolsets.security.kms.encryptionAlgorithm`

Encryption algorithm. Required for `azure`. AWS default: `SYMMETRIC_DEFAULT`.

#### `toolsets.security.encryption.algorithm`

Content encryption algorithm.

**Default:** `AES`

#### `toolsets.security.encryption.keySize`

Key size in bits (128, 192, or 256 for AES).

**Default:** `256`

#### `toolsets.security.encryption.cipherTransformation`

Cipher transformation string.

**Default:** `AES/GCM/NoPadding`

#### `toolsets.security.encryption.ivLengthBytes`

IV length in bytes (12 recommended for AES-GCM).

**Default:** `12`

#### `toolsets.security.encryption.gcmTagLengthBits`

GCM authentication tag length in bits.

**Default:** `128`

## Invitations and per-request keys

### `invitations.ttlInSeconds`

Invitation time to live in seconds.

**Default:** `259200` (72 hours)

```json
{
  "invitations": {
    "ttlInSeconds": 604800
  }
}
```

### `perRequestApiKey.ttl`

TTL for per-request API keys in seconds.

**Default:** `1800` (30 minutes)

```json
{
  "perRequestApiKey": {
    "ttl": 3600
  }
}
```

## Miscellaneous

### `apiKeyValidation.proxyCount`

Number of trusted proxies between client and DIAL Core. Used for `X-Forwarded-For` header parsing.

**Default:** `0`

```json
{
  "apiKeyValidation": {
    "proxyCount": 1
  }
}
```

### `asyncTaskExecutor.useVirtualThreads`

Use virtual threads for blocking tasks (Java 21+).

**Default:** `true`

```json
{
  "asyncTaskExecutor": {
    "useVirtualThreads": true
  }
}
```
