# Toolsets configuration

Toolsets provide MCP (Model Context Protocol) tool integrations for DIAL. They enable models to access external services, databases, APIs, and other tools during conversations.

Toolsets are defined in `config.json` under `toolsets.<toolset_name>`.

## Toolset attributes

### `endpoint`

The toolset API endpoint URL.

**Required**

```json
{
  "toolsets": {
    "web-search": {
      "endpoint": "http://mcp-server:5000/tools"
    }
  }
}
```

### `transport`

Transport protocol for communicating with the toolset.

| Value | Description |
|-------|-------------|
| `HTTP` | Standard HTTP request/response |
| `SSE` | Server-Sent Events for streaming |

**Default:** `HTTP`

```json
{
  "transport": "SSE"
}
```

### `displayName`

Human-readable name shown in DIAL Chat.

```json
{
  "displayName": "Web Search"
}
```

### `description`

Toolset description.

```json
{
  "description": "Search the web for current information"
}
```

### `iconUrl`

URL to the toolset's icon image.

```json
{
  "iconUrl": "https://cdn.example.com/icons/search.png"
}
```

### `userRoles`

Roles that can access this toolset.

```json
{
  "userRoles": ["premium", "admin"]
}
```

### `descriptionKeywords`

Keywords for discovery and matching.

```json
{
  "descriptionKeywords": ["search", "web", "internet", "browse"]
}
```

### `maxRetryAttempts`

Maximum retry attempts on failure.

```json
{
  "maxRetryAttempts": 3
}
```

### `allowedTools`

Available tools in the MCP server. Lists the specific tools users can invoke.

```json
{
  "allowedTools": ["search", "fetch_page", "summarize"]
}
```

### `forwardPerRequestKey`

Forward per-request API key to the toolset endpoint.

**Default:** `false`

> **Note:** Cannot be `true` when `authType` is `API_KEY`.

## Authentication

Toolsets support three authentication types with credential management via the `/v1/ops/toolset/signin` and `/v1/ops/toolset/signout` API endpoints.

### `authType`

Authentication type.

| Value | Description |
|-------|-------------|
| `NONE` | No authentication required |
| `API_KEY` | Static API key |
| `OAUTH` | OAuth 2.0 flow |

```json
{
  "toolsets": {
    "github": {
      "endpoint": "http://github-mcp:5000/tools",
      "authType": "OAUTH"
    }
  }
}
```

OAuth configuration is managed through static settings. See [Static settings](../settings-json/toolsets-security) for OAuth security configuration.

## Application type schemas

Application type schemas define custom application types with their own UI editors and endpoints. They enable schema-rich applications like Quick Apps and Code Apps.

Schemas are defined as a top-level `applicationTypeSchemas` array containing JSON schemas conforming to the meta-schema `https://dial.epam.com/application_type_schemas/schema#`.

### DIAL-specific schema properties

#### `dial:applicationTypeDisplayName`

Display name for the application type.

**Required**

```json
{
  "dial:applicationTypeDisplayName": "Quick App"
}
```

#### `dial:applicationTypeCompletionEndpoint`

Chat completion endpoint URL for applications of this type.

```json
{
  "dial:applicationTypeCompletionEndpoint": "http://quick-app-runner:5000/chat/completions"
}
```

#### `dial:applicationTypeResponsesEndpoint`

Responses API endpoint URL.

```json
{
  "dial:applicationTypeResponsesEndpoint": "http://quick-app-runner:5000/responses"
}
```

#### `dial:applicationTypeSchemaEndpoint`

JSON schema endpoint URL for dynamic schema generation.

```json
{
  "dial:applicationTypeSchemaEndpoint": "http://quick-app-runner:5000/schema"
}
```

#### `dial:applicationTypeEditorUrl`

URL to the custom editor UI for creating/editing applications of this type.

```json
{
  "dial:applicationTypeEditorUrl": "http://quick-app-editor:3000/edit"
}
```

#### `dial:applicationTypeViewerUrl`

URL to the viewer UI for viewing applications of this type.

```json
{
  "dial:applicationTypeViewerUrl": "http://quick-app-editor:3000/view"
}
```

#### `dial:applicationTypeIconUrl`

Icon URL for the application type.

```json
{
  "dial:applicationTypeIconUrl": "https://cdn.example.com/icons/quick-app.png"
}
```

#### `dial:applicationTypeInterceptors`

Interceptors applied to all instances of this application type.

```json
{
  "dial:applicationTypeInterceptors": ["quick-app-validator"]
}
```

## Examples

### Basic toolset

```json
{
  "toolsets": {
    "calculator": {
      "endpoint": "http://calc-mcp:5000/tools",
      "displayName": "Calculator",
      "description": "Perform mathematical calculations",
      "allowedTools": ["evaluate", "convert_units"]
    }
  }
}
```

### OAuth-authenticated toolset

```json
{
  "toolsets": {
    "github": {
      "endpoint": "http://github-mcp:5000/tools",
      "displayName": "GitHub",
      "description": "Access GitHub repositories and issues",
      "authType": "OAUTH",
      "descriptionKeywords": ["github", "repository", "code", "issues"]
    }
  }
}
```

### Quick App application type schema

```json
{
  "applicationTypeSchemas": [
    {
      "$id": "https://dial.epam.com/application_types/quick_app/v1",
      "$schema": "https://dial.epam.com/application_type_schemas/schema#",
      "dial:applicationTypeDisplayName": "Quick App",
      "dial:applicationTypeCompletionEndpoint": "http://quick-app-runner:5000/openai/deployments/quick-app/chat/completions",
      "dial:applicationTypeEditorUrl": "http://chat:3000/quick-app-editor",
      "dial:applicationTypeIconUrl": "https://cdn.example.com/quick-app.png",
      "dial:applicationTypeInterceptors": [],
      "type": "object",
      "required": ["instructions"],
      "properties": {
        "instructions": {
          "type": "string",
          "description": "System instructions for the Quick App"
        },
        "model": {
          "type": "string",
          "description": "Base model to use"
        }
      }
    }
  ]
}
```

### Multiple toolsets

```json
{
  "toolsets": {
    "web-search": {
      "endpoint": "http://search-mcp:5000/tools",
      "displayName": "Web Search",
      "transport": "HTTP"
    },
    "code-interpreter": {
      "endpoint": "http://code-mcp:5000/tools",
      "displayName": "Code Interpreter",
      "transport": "SSE",
      "userRoles": ["developer", "admin"]
    },
    "database": {
      "endpoint": "http://db-mcp:5000/tools",
      "displayName": "Database Query",
      "authType": "API_KEY",
      "userRoles": ["analyst", "admin"]
    }
  }
}
```
