# Developer features

Capabilities aimed at the people building on DIAL: extending the chat experience, enforcing guardrails, governing access, and measuring quality. Each demo shows the feature in action; the linked pages cover how to build it.

## Build a translator app

A look at building a simple text translator on DIAL — updating the configuration and wiring an application to talk to DIAL Core, using a local Ollama deployment.

https://youtu.be/xG7ilq9oeMA

## Custom buttons in apps

Action and Suggestion buttons extend the chat with clickable, interactive controls — how they look and how they fit into an application.

https://youtu.be/J0-Wati2E_Q

## Interceptors

Interceptors are DIAL's mechanism for Responsible AI and guardrails: middleware that modifies incoming or outgoing requests to enforce privacy and compliance policies.

https://youtu.be/h10E14kXlOI

## Roles and rate limits

Govern access for JWTs and API keys, and set usage limits per role — the controls behind DIAL's access and cost governance.

https://youtu.be/VWpn7M0bXaQ

## RAG Evaluation Toolkit

Three tools — a log parser, an evaluation library, and an evaluation UI — for validating the retrieval quality of any RAG-based application.

https://youtu.be/HYg_L2dxu6U
