# Apps and agents showcase

Real applications built on DIAL — from retrieval-augmented generation to multi-agent assistants, no-code apps, and interactive knowledge graphs. Each demo shows a finished capability, not how to build it.

## DIAL RAG

DIAL's retrieval-augmented generation: ask questions over your own PDFs, Office files, and text, with answers grounded in the retrieved content.

https://youtu.be/0fCWr4MB5KU

## DIAL Web RAG

RAG extended to the open web — combining cloud search with DIAL RAG, an example of DIAL's agentic approach where one app uses another as a building block.

https://youtu.be/eYrrCegf-SI

## ChatHub

A multi-agent orchestrator that routes each prompt to the right tool — text, image, vision, code interpreter, or RAG — behind a single chat entry point.

https://youtu.be/IG0HawQuU-w

ChatHub 2.0 introduces a prompt routing engine that picks the best tool for each request and chains agents for efficient, combined responses.

https://youtu.be/8Npbd0rESPI

## Code Apps

Write, deploy, and run Python applications directly in the DIAL UI — no separate hosting to manage — demonstrated by building a simple RAG app end to end.

https://youtu.be/oysFwinnox4

## Quick Apps

The no-code agent orchestrator: combine MCP tool sets, AI agents, and models into multi-agent workflows without writing code.

https://youtu.be/80VkNaScYX4

A Quick App that returns weather information and clothing recommendations for any city.

https://youtu.be/Y7xbH6088EI

Using attached files as a RAG source inside a Quick App.

https://youtu.be/0EUx4kw0ssc

## MCP integration

Connect DIAL to external systems through the Model Context Protocol, surfaced through the Admin Panel and Quick Apps.

https://youtu.be/asv6F6i_vww

## StatGPT

An open-source "talk to your data" platform for official statistics, built on DIAL — querying, analyzing, and visualizing statistical data through natural language with an agentic architecture.

https://youtu.be/xoJIdFllZgU

## OMICS AI Assistant

A multi-agent application for scientific data: specialized agents handle query, visualization, and analysis over an OMICS dataset, with results shareable across a team.

https://youtu.be/5Q2XUwx5hYw

## Talk-to-your-data apps

Two case studies built on DIAL's talk-to-your-data framework, where dedicated agents handle each step behind a simple natural-language interface.

The Athlete Profile Generator builds a detailed profile from just a player's name.

https://youtu.be/ujcPHldA7iI

The Oil and Gas AI Assistant explores production data, generating code for visualizations, calculations, and interpretation.

https://youtu.be/mQDorqSvGZE

## DataViz App

A code walkthrough of the agent-orchestration app that visualizes and analyzes financial data — the precursor to StatGPT.

https://youtu.be/70wb8qzkLMw

## Entity Extractor

Identify and pull predefined information from unstructured documents — for example, a claims-processing assistant that extracts, validates, and reports on insurance claim data.

https://youtu.be/9rBfa6Ix-is

## TimeBase AI Assistant

A multi-agent assistant that retrieves, analyzes, and monitors time-series financial data in real time — generating indicators, trading signals, and live alerts, with each agent's reasoning on screen.

https://youtu.be/fqFx4vXbjLM

## Mind Map Studio

Turn documents and data sources into an interactive knowledge graph you can explore in natural language.

https://youtu.be/XYZfWeGdFcE

Build and publish a Mind Map yourself with the no-code Studio editor.

https://youtu.be/PpCEatAM_ak

## Caspase-2 article visualization

A Mind Map example that turns dense scientific articles into a navigable, hierarchical knowledge graph with summaries and source links.

https://youtu.be/SylMQwIBpGA

## Visualization of a cinematic universe

A Mind Map that maps the Lord of the Rings universe — extracting and connecting data from fan wikis, enriched with AI-generated imagery and explorable through chat.

https://youtu.be/8fPFrJRXn7Y

## Guided Conversations

An interactive-survey application built on DIAL that walks a user through a guided, structured conversation.

https://youtu.be/1XT4NDh1xnc
