# Platform overview

What DIAL is, at a glance: an enterprise-grade platform for building, running, and governing GenAI applications — and the single API that ties every model and app together.

## What is DIAL

A two-minute introduction to the platform: what problem it solves and where it fits in your GenAI stack.

https://youtu.be/I3Ca8bBIg3M

## DIAL as an application server and orchestration platform

See how DIAL acts as a unified GenAI application server, marketplace, and collaboration layer — with low- and no-code tools to create and share apps, and no vendor lock-in across clouds and environments.

https://youtu.be/Ud2UyXjNK4I

## One Unified API for every model and app

Every model and application in DIAL is reachable through one [Unified API](https://dialx.ai/dial_api). This removes per-integration protocols and the risk of vendor lock-in: swapping a model or app behind your application does not force a rewrite.

https://youtu.be/PqIHlRKQMkw
